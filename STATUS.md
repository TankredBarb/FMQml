# FM Project Status & Roadmap

## ✅ Что реализовано (Current State)

### 1. Ядро и Производительность (C++/Qt)
- **Asynchronous Scanning:** Поиск и чтение файлов вынесены в фоновый поток (`DirectoryScanner`). UI не блокируется даже на папках с 10k+ файлов.
- **Batch Updates:** Модель получает данные пачками по 100 элементов, список начинает наполняться мгновенно.
- **Incremental Updates:** Реализовано "умное" обновление модели. При изменениях в папке (создание/удаление) обновляются только нужные строки, что позволяет работать QML-анимациям.
- **Sorted Insertion:** Новые файлы и папки автоматически вставляются в нужное место (сначала папки, затем файлы, по алфавиту), сохраняя порядок без полного сброса списка.
- **File Watching:** Интеграция `QFileSystemWatcher`. Список обновляется автоматически при изменениях в текущей директории.
- **Operation Queue:** Очередь асинхронных операций с поддержкой копирования, перемещения и удаления.
- **Conflict Resolution:** Система разрешения конфликтов (Replace/Skip/Keep Both) с поддержкой "Apply to all".

### 2. Интерфейс и UX (QML)
- **Multi-Panel:** Двухпанельный режим с синхронизацией фокуса и переключением Split/Unsplit.
- **View Modes:** Плавное переключение между Списком (List) и Сеткой (Grid).
- **Smooth Animations:** Использование `displaced`, `add` и `remove` переходов в списках. Элементы плавно раздвигаются и схлопываются.
- **Native Icons:** Реализован `IconProvider`, вытягивающий реальные иконки из системы (Windows/Linux).
- **Thumbnails:** `ThumbnailProvider` для отображения миниатюр изображений в режиме сетки.
- **Smooth Scrolling:** Инерционный скроллинг и эффект "пружины" (`overshoot`) на границах списков.
- **Theme Engine:** Динамическая смена Светлой/Темной темы.

### 3. Инструменты и Навигация
- **Undo/Redo:** Базовый механизм отмены и повтора для операций копирования и перемещения.
- **Keyboard Friendly:** Навигация стрелками, Enter, Backspace, Ctrl+C/V/X, Ctrl+Z/Y.
- **Smart Creation:** Кнопка "+ Folder" создает папки с уникальными именами ("New Folder (1)"), предотвращая ошибки записи.

## 🚀 Мысли по развитию (Future Ideas)

### 1. "Total Smoothness" (Идеальная плавность)
- **Shared Element Transitions:** Плавный "перелет" иконок при переключении между режимами List и Grid.
- **Pre-fetching:** Фоновое сканирование соседних папок и часто посещаемых мест для мгновенного перехода.
- **Input Prediction:** Визуальный отклик на нажатия (ripple effect), чтобы каждое действие "чувствовалось".

### 2. Функциональное расширение
- **Search Everywhere:** Полнотекстовый поиск по файлам с использованием фонового индексатора.
- **Tabs Support:** Поддержка вкладок внутри каждой панели, как в современных браузерах.
- **Quick Look:** Быстрый просмотр содержимого файла по нажатию пробела (текст, код, картинки).
- **Archive Integration:** Работа с ZIP/7Z архивами как с обычными папками.

### 3. Системная интеграция
- **Context Menu Native:** Интеграция системного контекстного меню (Windows Shell).
- **Properties Dialog:** Полноценное окно свойств файла/папки с графиками распределения места.
- **Cloud Integration:** Отображение статуса синхронизации для OneDrive/Dropbox.
